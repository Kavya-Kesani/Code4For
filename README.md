# Career Re-Entry Assistant for Women

An AI-powered platform that helps women confidently return to professional careers after taking breaks for motherhood or caregiving responsibilities.

## Features

- **Profile Creation**: Capture previous work experience, career break duration, and desired career path
- **AI-Powered Analysis**: Get personalized skill gap analysis based on current industry requirements
- **Learning Roadmap**: Receive tailored recommendations for certifications and courses
- **Job Suggestions**: Discover suitable returnship programs and job opportunities
- **Interview Preparation**: Get guidance for skill refresh and interview preparation

## Tech Stack

- **Frontend**: React.js with Material-UI
- **Backend**: Node.js with Express.js
- **Database**: MongoDB
- **AI**: OpenAI GPT-3.5 Turbo

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB
- OpenAI API Key

### Installation

1. Clone the repository
2. Install dependencies for both client and server:

```bash
npm install
npm run install-server
npm run install-client
```

3. Set up environment variables:
   - Copy `server/.env` and update with your MongoDB URI and OpenAI API key

4. Start the development servers:

```bash
npm start
```

This will start both the React client (port 3000) and Express server (port 5000).

## Usage

1. Register for an account
2. Complete your career profile
3. View your personalized AI analysis and recommendations
